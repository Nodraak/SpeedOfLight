# Speed of light

*Reenactment of experiment(s) to measure the speed of light.*

Following a [video by AlphaPhenix in 2018](https://www.youtube.com/watch?v=YMO9uUsjXaI), where he redo Fizeau's experiement, I set out to reenact the experiement myself.
After some research, it seems to me that Foucault's experiement is easier to perform and will give a more precise value, while using a somehow similar technique.
